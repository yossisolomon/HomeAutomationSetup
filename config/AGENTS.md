# config/ — Home Assistant Configuration

## Structure

```
config/
  configuration.yaml   # main HA config entry point
  secrets.yaml         # GITIGNORED — never commit
  template/
    fans.yaml          # GITIGNORED — generated by scripts/sync_rf_codes.py
  automations.yaml
  scripts.yaml
  ...
```

## Rules

- **Never commit `secrets.yaml`** — use HA Secrets for all API keys, tokens, passwords.
- **`config/template/fans.yaml` is generated** — do not edit manually. Run `sync_rf_codes.py --restart` on blacky to regenerate from `scripts/rf_codes_cache.json`.
- Prefer UI-managed integrations over manual YAML where possible.
- After editing YAML, validate in HA: **Developer Tools → YAML → Check Configuration**.
